## Problem 2

#### Use the `list.files` to create datafrane of file names

``` r
arms_df = 
  tibble(
    file_path = 
           list.files(path = "data", full.names = TRUE)
    ) %>% 
  mutate(data = map(file_path, read_csv)) %>% 
  unnest()
```
